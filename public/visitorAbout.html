About IskolarLink

IskolarLink is a scholarship management and communication system for the Iskolar ng Koronadal – Kabugwason Paglaum Program.

Core Features

  • Publish announcements & events
  • Provide application requirements
  • Allow scholars to sign up and upload documents
  • Notify scholars about tasks and deadlines

How to Apply

Review the Application Requirements, then create an account using Sign up.
